# Nobretech iPhone Assets

Imagens organizadas para uso no sistema NOBRETECH STORE.

Estrutura:
`/public/product-assets/apple/iphone/{modelo}/{arquivo}.webp`

Regra:
- 1 arquivo = 1 modelo + 1 cor
- nomes em minúsculo
- sem espaço
- sem acento
- separados por hífen

Também foi gerado `asset-manifest.json` com o mapeamento original -> novo caminho público.
